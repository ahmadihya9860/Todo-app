تطبيق مدير المهام (ToDo App) هو أداة بسيطة تهدف إلى مساعدة المستخدمين على تسجيل واجباتهم اليومية وتنظيمها. يتميز التطبيق بواجهة واضحة تعرض قائمة بالمهام، حيث يمكن للمستخدم إضافة مهمة جديدة بضغطة زر، أو شطب المهمة عند الانتهاء منها، أو حذفها عند الرغبة.

المكونات الأساسية للتطبيق:

قائمة المهام: تعرض العناوين بشكل مرتب وسهل القراءة.

إضافة سريعة: حقل نصي لإدخال المهام الجديدة فوراً.

علامة الإنجاز: مربع اختيار (Checkbox) لتمييز المهام المكتملة.

الحذف السريع: إمكانية إزالة المهام التي لم تعد مطلوبة.

هذا التطبيق يعد مشروعاً مثالياً للمبتدئين في Flutter لأنه يركز على كيفية عرض القوائم والتعامل مع المدخلات بشكل مباشر.
